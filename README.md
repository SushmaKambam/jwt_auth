# MongoDB Authentication System

A complete authentication system using Node.js, Express, MongoDB, and JWT.

## Features
- User Registration
- User Login
- JWT Authentication
- Protected Routes
- MongoDB Integration
- Error Handling
- Modern UI Design
- Input Validation
- Secure Password Hashing

## Prerequisites
- Node.js installed
- MongoDB Atlas account
- Git installed
- Text editor (VS Code recommended)
